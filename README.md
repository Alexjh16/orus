## Desarrollo

```bash
# Instalar dependencias
flutter pub get

# Ejecutar en Linux
flutter run -d linux

# Ejecutar en Chrome
flutter run -d chrome

# Ejecutar en dispositivo Android/iOS
flutter run
```
